


<!-- Append vs AppendChild -->
1.
Can append multiple nodes or strings
Can append only one node

2.
append can have multiple argument
AppendChild have only one argument

3.
append() → Returns undefined.
appendChild() → Returns the appended node

<!-- remove vs removeChild -->
1.
call on The element itself
call on The parent element

2.
remove() → Best for modern, direct removal.
removeChild() → Best for controlled DOM operations .

3.
remove() → Returns undefined.
removeChild() → Returns the removed node.
